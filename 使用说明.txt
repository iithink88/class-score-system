========================================
  通用班级学情提分系统 - 使用说明
========================================

【系统简介】
本系统面向中小学全学科教师，支持自动批改作业/试卷、
学情统计分析、个性化报告生成、IMA知识库联动等功能。
无需学生安装APP，仅需老师电脑+实物展台/手机拍照即可使用。

========================================
一、快速开始（5步完成）
========================================

第1步：双击 start.bat 启动系统
  → 浏览器自动打开操作界面（http://127.0.0.1:8099）
  → 如未自动打开，手动在浏览器输入上述地址

第2步：点击「设置」标签，配置API密钥
  → OCR识别：填写通义千问 DashScope API Key（sk-开头）
  → AI批改：填写火山方舟 API Key 和 推理接入点ID
  → IMA知识库：填写 Client ID 和 API Key
  → 点击「保存所有设置」

第3步：点击「学生管理」标签，导入学生名单
  → 在文本框中粘贴名单（格式：学号,姓名，每行一个）
  → 或从文件导入（选择txt文件）
  → 也可逐个添加学生

第4步：点击「作业上传」标签，上传作业图片
  → 选择学生
  → 拖拽或点击上传区域选择图片
  → 支持同一学生上传多张图片

第5步：点击「批改分析」标签，开始批改
  → 点击「开始批改分析」按钮
  → 等待分析完成（查看进度和日志）
  → 点击「生成班级总体报告」生成班级汇总

========================================
二、功能详解
========================================

【学生管理】
- 批量导入：支持txt格式名单，每行格式「学号,姓名」
- 单个添加：输入学号和姓名
- 删除：点击删除按钮
- 清空：一键清空所有数据

【作业上传】
- 支持JPG/PNG格式
- 自动校正图片方向（基于EXIF）
- 自动压缩（宽度超2000px自动缩放，JPEG 85%质量）
- 支持批量上传，同一学生可上传多张

【批改分析】
- OCR识别：通义千问识别印刷题目和手写答案
- AI批改：火山方舟Doubao逐题判分
- 错题分类：概念混淆/计算失误/审题不清/步骤遗漏/粗心大意/知识盲区
- 知识点标注：自动标注每道题涉及的学科知识点
- 个人报告：每个学生一份，含逐题批改、错题分析、学习建议、练习题
- 班级报告：全班汇总，含正确率统计、易错题排行、教学建议

【报告查看】
- 所有报告自动保存到桌面的「分析_YYYYMMDD_HHMMSS」文件夹
- 个人报告文件名：学号_姓名.txt
- 班级报告文件名：班级总体学情分析.txt
- 可在Web界面直接查看报告内容
- 一键打开报告文件夹

【IMA同步】
- 将作业图片和学情报告批量上传到IMA共享知识库
- 学生通过IMA对话即可查询个人错题和薄弱点
- AI自动生成变式练习题
- 老师可调取全班学情数据

【设置】
- 所有API密钥可视化填写，自动保存
- 支持测试每个API连接是否正常
- 可修改年级、班级、学科等基本信息

========================================
三、API密钥获取指南
========================================

【通义千问 DashScope API Key（OCR识别用）】
1. 用淘宝/支付宝登录阿里云，完成实名认证
2. 打开 https://dashscope.console.aliyun.com/
3. 左侧菜单找到「API-KEY管理」
4. 点击「创建API-KEY」
5. 复制生成的 sk-xxx 密钥

【火山方舟 ARK API Key（AI批改用）】
1. 打开 https://console.volcengine.com/ark/
2. 创建API Key
3. 在「在线推理」中创建推理接入点
   → 模型选择：doubao-seed-2.0-pro
   → 复制接入点ID（ep-开头）
4. 填入系统的设置页面

【IMA知识库凭证（IMA同步用）】
1. 安装IMA客户端：https://ima.qq.com
2. 登录后点击左上角头像 → Claw配置
3. 复制Client ID和API Key
4. 填入系统的设置页面
5. 注意：凭证需定期续期，续期入口 https://ima.qq.com/agent-interface

========================================
四、常见问题
========================================

Q：启动后浏览器没有打开？
A：手动在浏览器输入 http://127.0.0.1:8099

Q：OCR识别失败？
A：1. 检查API Key是否正确 2. 确保图片清晰 3. 检查网络连接

Q：AI批改报错？
A：1. 检查ARK API Key 2. 确认推理接入点ID正确 3. 检查模型是否为doubao-seed-2.0-pro

Q：IMA同步失败？
A：1. 检查IMA凭证是否过期 2. 确认知识库名称正确 3. 到ima.qq.com/agent-interface续期

Q：图片上传后分析很慢？
A：正常现象，每张图片需要先OCR再AI批改，约10-30秒/张

Q：可以离线使用吗？
A：可以离线使用Web界面和学生管理功能，但OCR和AI批改需要网络

Q：如何分享给其他老师？
A：将整个文件夹复制给对方，对方只需修改API密钥即可使用

Q：支持哪些学科？
A：所有学科均支持，在设置中修改学科名称即可。系统会根据学科自动调整批改逻辑。
